# Ocean Cloud

Ocean Cloud는 자료를 수집하고, 저장된 데이터를 다양한 방식으로 조회할 수 있도록 만든 웹 플랫폼입니다.  
서버 정의 기반 입력 화면, 데이터 탐색 화면, 지도 시각화, 차트, CSV Export 같은 기능을 하나의 흐름으로 연결해 운영할 수 있도록 구현했습니다.

이 프로젝트에서는 단순히 입력 화면 하나를 만드는 데 그치지 않고, 자료를 등록하고 조회하는 흐름 전체를 다뤘습니다.  
프론트엔드 화면 구성부터 상태 관리, API 설계 및 구현, 데이터 저장 구조, 시각화 기능까지 함께 연결하며 웹 서비스 전반을 구현한 프로젝트였습니다.

## 핵심 내용

- 서버 정의 기반 동적 입력 화면 구현
- 다양한 조건으로 데이터를 조회할 수 있는 탐색 기능 구현
- 지도 시각화, 차트, CSV Export, 위치 기반 기능 제공
- React, Express, MongoDB 기반으로 프론트엔드와 백엔드 모두 개발

## 제 역할

책임급 개발자 1명과 2인으로 진행했습니다.
동적 입력 시스템, 데이터 탐색 기능, 상태 관리, API 설계 및 구현, MongoDB 연동까지 웹 서비스 전반을 담당했습니다.
입력과 조회, 시각화 기능이 서로 따로 놀지 않고 하나의 서비스 흐름으로 이어지도록 정리하는 데 집중했습니다.

## 결과

- 동적 입력 화면 최초 3개 → 10개 이상으로 확장
- 서버 정의 변경 시 프론트엔드 화면 수정 없이 대응
